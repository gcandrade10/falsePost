# falsePost
Generate fake user data and make unlmited posts to common uniandes phishing sites 

## How to get it working
### Install "request" package dependency ⬇️
```shell
$ npm install request
## or
$ yarn add request
```
 
### Try a single post 1️⃣
 
```shell
$ node falsepost.js
```


### Start Infinite Posting ∞🦄👌
```shell
$ while true
do
node falsepost.js
done

## or

$ sh start.sh
```
