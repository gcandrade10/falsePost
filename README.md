# falsePost
A tool to make unlimited posts to regular uniandes phishing sites 

## How to get it 
### Install "request" package dependency ⬇️
```shell
npm install request
## or
yarn add request
```
 
### Try a single post 1️⃣
 
```shell
node falsepost.js
```


### Start Infinite Posting ∞🦄👌
```shell
$sh start.sh
```
